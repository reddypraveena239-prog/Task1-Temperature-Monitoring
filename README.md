# Task 1 - Temperature Monitoring & LCD Display

## Objective

To monitor temperature using a TMP36 temperature sensor and display the measured value on a 16x2 LCD display. The temperature readings are also logged through the Serial Monitor.

## Components Used

* Arduino UNO
* TMP36 Temperature Sensor
* 16x2 LCD Display
* Breadboard
* Jumper Wires

## Software Used

* Tinkercad Circuit Simulator
* Arduino IDE

## Working Principle

The TMP36 sensor measures the surrounding temperature and produces an analog voltage output. Arduino reads this voltage through an analog input pin, converts it into temperature values in degrees Celsius, displays the value on the LCD, and logs the readings through the Serial Monitor.
